# les gouvernements de Paul Biya

Travail de recherche sur les différents remaniements et réorganisations de Paul Biya, Président de la république du Cameroun, depuis le 06 Novembre 1982 à nos jours.

Les éléments utilisés dans le cadre de cette recherche ont été compilés en dossiers:

1. Le dossier "Réorganisation" comprend les décrets relatifs aux réorganisations ie la composition du gouvernement en terme de département ministériels.

2. Le dossier "Remaniement" est composé des décrets relatifs aux changements à la tête des départements ministériels, communément appelé "Jeu de chaises musicales".

3. Le dossier "Éditions de Cameroon Tribune" est quant à lui composé des éditions de Cameroon Tribune achetés au Kioske de la SOPECAM lorsque celui-ci était encore disponible.

4. Le dossier "Biographie des Ministres" recense tous les documents trouvés nous permettant d'établir un CV express des ministres du Renouveau.

5. Le fichier "gouvernements.xlsx" contient toutes les données regroupées sous fichier Excel.
